To jar the program from Eclipse:

1.       Make sure you have updated to the latest version of the project

2.       Go to File > Export

3.       Select Java > JAR file and click next

4.       Select the project (c560aa08), set the export destination, and click next

5.       Click finish

6.       The jar file will now be in the specified directory

To run the jar file:

1.       Make sure that java is installed for the directory that you are running this from

2.       Simply double click on the jar file that you created
